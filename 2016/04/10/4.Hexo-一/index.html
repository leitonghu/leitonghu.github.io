Hexo(一)

Posted on 2016-04-10   |   In Hexo   |  

前期准备

认识Hexo

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。
hexo.io

安装GIT

GitHub Windows
简单可依赖，安装完成后依据提示操作即可

安装Node.JS

node.js
注意 安装完成后添加Path环境变量，使npm命令生效。新版已经会自动配置Path

安装Hexo

配置好GitHub目录后，双击桌面上的Git Shell，输入npm命令即可安装
npm install hexo-cli -g
npm install hexo –save

Hexo初始化配置

创建Hexo文件夹

建立自己的目录：H:\hexo\
右键进入gitshell

hexo init

进行初始化

新建完成后，指定文件夹的目录如下

├── _config.yml
├── package.json
├── scaffolds
├── scripts
├── source
| ├── _drafts
| └── _posts
└── themes

安装Hexo插件

npm install hexo-generator-index --save
npm install hexo-generator-archive --save
npm install hexo-generator-category --save
npm install hexo-generator-tag --save
npm install hexo-server --save
npm install hexo-deployer-git --save
npm install hexo-deployer-heroku --save
npm install hexo-deployer-rsync --save
npm install hexo-deployer-openshift --save
npm install hexo-renderer-marked@0.2 --save
npm install hexo-renderer-stylus@0.2 --save
npm install hexo-generator-feed@1 --save
npm install hexo-generator-sitemap@1 --save

查看本地

hexo server

Hexo 简写命令

hexo n #生成文章，或者soruce_posts手动编辑
hexo s #本地发布预览效果
hexo g #生成静态文件

部署静态网页到GitHub

注册设置GitHub

  1. 登录GitHub，注册自定义用户名如leitonghu
  2. 在主页右下角创建New repository，name必须和用户名一致如leitonghu.github.i表项
  3. 首次创建耐心等待10分钟左右审核，之后即可访问静态主页如http://leitonghu.github.io

遇到问题

1.提示，”ERROR Deployer not found: github”
解决办法：
a.安装 npm install hexo-deployer-git –save
b.将deploy 的 type由github改为git
